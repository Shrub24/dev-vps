# Bifrost Docker image reference

Official examples use the canonical image repository/name:

- `maximhq/bifrost`

Versioned examples documented in the repo/docs include:

- `maximhq/bifrost:v1.3.0`
- `maximhq/bifrost:v1.3.9`
- `maximhq/bifrost:v1.4.21`
- `maximhq/bifrost:v1.4.22`

Architecture-specific tags are documented for at least one version:

- `maximhq/bifrost:v1.3.9-amd64`
- `maximhq/bifrost:v1.3.9-arm64`

## Exact example strings

- `docker run -p 8080:8080 maximhq/bifrost`
- `docker pull maximhq/bifrost`
- `docker pull maximhq/bifrost:v1.3.9`
- `docker pull maximhq/bifrost:v1.3.9-amd64`
- `docker pull maximhq/bifrost:v1.3.9-arm64`

## Conclusion

Yes — versioned arm64 tags are documented in the official quickstart docs.
